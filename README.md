# wp-douban

可以以豆列的方式展示电影、图书、音乐和相册。

图片内容缓存到本地解决豆瓣防盗链。

读取 API 数据缓存时间为 1 个月。

###

WPD_CACHE_IMAGE 设置为 false 可以关闭图片缓存

### 插入方式

直接在文章中粘贴豆瓣 url 即可。

相册展示

<pre data-type="shortcode">https://www.douban.com/photos/album/1693226620</pre>
